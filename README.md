# edu_ciaa_baremetal
Baremetal applications.

baremetal_blinking --> parpadeo del led con retardo por bucle (usando inline en asm)
baremetal_blinking_timer_irq --> parpareo del led usando timer e interrupciones
baremetal_adc_timer_irq --> conversión analógica-digital usando timer e interrupciones
baremetal_dac_timer_irq --> conversión digital-analógica usando timer e interrupciones (rampa ascendente de 0 a 3,3V en 10,24segundos)
